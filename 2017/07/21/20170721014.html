CentOS修改IP地址

发表于 2017-07-21 分类于 Tool

CentOS 修改IP地址

  • 修改IP都需要找到网卡配置文件
    /etc/sysconfig/network-scripts/ifcfg-eth0
  • 都需要进行DNS配置
    /etc/resolv.conf
  • 关闭防火墙
    1. 关闭命令 service iptables stop
    2. 永久关闭 chkconfig iptables off

CentOS 6.0版本修改方式

执行命令 service network restart 重启网络

CentOS 7.0版本修改方式

  • 执行命令 systemctl restart network.service 重启网络
  • 执行 ip add 查看是否配置正确
